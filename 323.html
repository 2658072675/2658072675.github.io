比notepad++好用！Sublime Text在Linux下的安装与配置

发表于2021-05-09|更新于2024-09-30|玩法开启

在转到Ubuntu平台之后，许多环境重新进行了配置，回过头来想写篇稿子却发现自带的文本编辑器太简单了，功能不够丰富。于是乎，准备给我心爱的电脑安装上我在win上常用的文本编辑其notepad++，刚准备百度其官网，就发现了这样一件事

好奇心驱使下，我点开了这个关键词，之后找到了CSDN上的一篇帖子，**_notepad++事件_卸载、抵制Notepad++_**，了解了这件事的真相。

原文链接：**_点我_**

大家点开就能看到了，真搞不明白，一个文本编辑其而以，这么普通却有那么自信！敢作出分裂我国的事情，反正我是不会再用这个软件了，CSDN作者也给我们推荐了其他可替代的软件，我省的大家麻烦就直接贴出来了。

  • VSCode  https://code.visualstudio.com/
  • EditPlus  https://www.editplus.com/
  • UltraEdit-32 https://www.ultraedit.com/
  • Sublime Text https://www.sublimetext.com/
  • Source Insight  https://www.sourceinsight.com/

这些软件我大致看了一下，选取了一个比较适合为习惯和审美的，也就是这篇文章的主角Sublime Text，接下来就一步一步安装

Ubuntu打开终端，从上往下逐条输入执行就可完成安装，打开一看，居然是英文的，Emmm…

好吧，那就设置一下中文吧，百度一下发现，要切换成中文需要下载中文包。

然而网上都说第一步先在Preferences里面找到Package Control，但是当时我的Preferences里面并没有这一项，顿时人傻了。

再查找一番后发现，Package Control是需要自己手动安装的（为什么网上的教程没有呢，难道是新版本需要手动安装旧版本不需要？），算了无所谓了，能安装就行了，下面就是我整理的安装方法。

打开Sublime Text,使用快捷键Shift+Ctrl+P,弹出查找栏，在搜索框中输入关键字 install ,点击选择其中的：Package Control: Install Package

点击之后稍微等一小会儿，安装完成会弹出如下界面

在搜索框中输入chinese，选择下拉框中的ChineseLocalizations

点击之后，中文包就安装完成了，之后关闭软件重新打开，点击任务栏中的Help->Language->简体中文 ，中文切换就完成了！

之后的个性化设置就自己去摸索吧！
